# SGI-USA Caribbean Region — Interactive Contact Map

An interactive multilingual map connecting visitors with SGI-USA contacts across 21 Caribbean territories.

## Features

- **Interactive SVG map** — hover to identify, click to contact
- **6 languages** — English, Spanish, French, Haitian Creole, Papiamentu, Dutch
- **Contact form** — territory-specific email routing with CC to regional team
- **Responsive** — works on desktop and mobile
- **Zero dependencies** — single HTML file, no build step

## Deploy

Push to GitHub and connect to [Vercel](https://vercel.com) — or run locally:

```
npx serve .
```
